2. 1주차 주요 개념 키워드 적어보기
- 버전관리 : 변화에 대한 상태 정리(편집자, 시간, 바뀐내용)
- git 초기화 : git으로 관리하도록 설정
- 커밋(commit) : 변화한거 확인용,현재 프로젝트 상태
- pull : 코드 떙겨오는거
- push : 올리는거
- add(=staging) : commit 나눠서 하도록
- clone : 쓸려고 불러오는거
- tracking : 동기화 같은 
- 로컬repo : 내가쓰는거 /원격repo: 클라우드 느낌
